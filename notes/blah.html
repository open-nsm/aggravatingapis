API

The Blah API is developed by Blah Blah and allows users to do blah.

Useful

Blah API implements all the features of the product Blah which makes it very useful.

Conventions

Blah API does not follow consistent conventions. The case of alphabetic characters in the responses change every request.

Documentation

Blah API's documentation is outdate. A few of the API endpoints do not exist and in some calls the required parameters are not listed.

JSON

The API uses XML for the query language.

Rating

Based on each criteria we give this plugin a rating of 1/5.
