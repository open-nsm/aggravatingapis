Purpose

We work with many different API's on a daily basis, and we find that many of them could be improved (especially those from the information security space). It's as if API development is an afterthought, possibly contracted out to others who complete them hastily and haphazardly. The purpose of this website is to rate the API's we depend on, so we can motivate change through tough love and help other developers. In similar fashion to the Two Factor Auth list.

Contributing

Add an API to the list by submitting a pull request to the gh-pages branch on the GitHub repo. We require evidence to support the addition of an API; A HTML page should be added to the notes directory and linked from the notes column of the table.
