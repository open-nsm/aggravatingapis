Purpose

We work with a lot of different API's on daily basis and we find that many of them could be improved, especially those from the information security space. It's as if API development was a second thought, say, where a college student was contracted hastily to just get it done. With that said, the purpose of this website is to rate API's of useful services we depend on, with some tough love we can motivate change and help other developers. In similar fashion to the Two Factor Auth list.

Contributing

Add an API to the list by submitting a pull request to the gh-pages branch on the GitHub repo. We require evidence to support the addition of an API; A HTML page should be added to the notes directory and linked from the notes column of the table.
